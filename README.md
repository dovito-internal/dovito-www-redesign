# Dovito WWW Redesign

Marketing website for [Dovito Business Solutions](https://dovito.com) — operations consulting for construction companies.

Redesigned with a modern, sanity.io-inspired aesthetic. Built with React, Vite, Tailwind CSS, and shadcn/ui.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/` — deploy to any static host.
